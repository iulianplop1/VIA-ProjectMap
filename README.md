# VIACompass

An interactive campus map for VIA University College.

## Why I Built This
I built this during a Hackathon at VIA University College. As a student, I experienced firsthand how confusing the campus layout can be. New students constantly get lost trying to find their classrooms, the MakerSpace, or just figuring out what's on the menu at the canteen.

I wanted to solve this real problem by creating a fast, mobile-first campus map. 

## Features
- **Interactive Map:** Switch between floors and navigate the campus easily.
- **Room Availability:** See in real-time which rooms are free and which are occupied.
- **Point-to-Point Directions:** Search for a room and get clear directions on how to reach it.
- **Campus Services:** Check today's canteen menu and the status of the 3D printers in the MakerSpace.
